# API Documentation

## Auth
- `POST /api/auth/login`

## Jemaat
- `GET /api/jemaat`
- `POST /api/jemaat` (auth)
- `PUT /api/jemaat/:id` (auth)
- `DELETE /api/jemaat/:id` (auth)
