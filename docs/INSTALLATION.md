# Installation Guide

## 1. Clone repository
```bash
git clone <repo-url>
cd cms-gereja
```

## 2. Backend setup
```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

## 3. Frontend setup
```bash
cd ../frontend
cp .env.example .env
npm install
npm run dev
```

Frontend: http://localhost:5173  
Backend: http://localhost:5000
