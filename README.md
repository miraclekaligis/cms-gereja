# 🏰 Church Information System (CMS) - Version 1

Sistem Informasi Gereja dengan CMS yang memungkinkan pengurus gereja mengelola data jemaat, jadwal ibadah, pengumuman, dan konten website tanpa perlu coding.

## 🎯 Features

### 🟢 Website Publik
- **Beranda** - Homepage dengan hero section yang editable
- **Jadwal Ibadah** - Daftar lengkap kegiatan ibadah
- **Pengumuman** - Berita dan pengumuman terbaru
- **Tentang Gereja** - Informasi gereja yang bisa diedit

### 🔵 Admin Dashboard
- **Login System** - Autentikasi admin dengan JWT
- **Kelola Jemaat** - CRUD data anggota jemaat
- **Kelola Jadwal** - CRUD jadwal ibadah dan kegiatan
- **Kelola Pengumuman** - CRUD pengumuman/berita
- **Edit Halaman** - CMS editor untuk halaman website
- **Dashboard Overview** - Statistik dan ringkasan
